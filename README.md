I like building in the new agent era, but I still care about the old fundamentals: good product instinct, clean systems, sharp interfaces, and code that survives contact with other people.

Lately that means a lot of TypeScript, React, Python, data plumbing, AI-assisted workflows, dashboards, observability, and small weird tools that probably started as "this will take ten minutes."

Before the current wave, I co-created Dockwell, a lightweight Docker monitoring/debugging tool, and spent time thinking about microservices, DX, and how to make systems easier to operate.
